## Run Locally

Clone the project

```bash
  git clone https://github.com/PJPhukan/zidio-eCommerce
```

Go to the project directory

```bash
  cd zidio-eCommerce
```

Install dependencies (Client side)

```bash
  cd client
  npm install
```

Install dependencies (Server side)

```bash
  cd server
  npm install
```
